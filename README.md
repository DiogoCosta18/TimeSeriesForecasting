# Time Series Forecasting Paper Results

This repository contains derived result artifacts for the paper experiments on
seasonal decomposition in time series forecasting.

Included:

- Aggregate benchmark result tables from `results/`.
- Run metadata and compact derived feature manifests from
  `results/run_20260525T112121/`.
- Paper analysis tables, export reports, and figures from `results_week4/`.

Excluded:

- Forecasting pipeline code.
- Vast.ai machine setup files.
- Raw M3/M4 datasets.
- Notebooks, caches, checkpoints, and runtime folders.
- Personal information, credentials, API keys, and environment files.
